Politique de confidentialité

Cette politique de confidentialité décrit comment nous recueillons, utilisons et protégeons les informations personnelles que vous nous fournissez lorsque vous utilisez notre application web.

Collecte des données

Nous collectons les données suivantes :

  • Votre adresse e-mail : nous l'utilisons pour la connexion et pour vous envoyer des informations relatives à notre application.
  • Le type de voiture que vous possédez : nous l'utilisons pour personnaliser votre expérience utilisateur.
  • La plaque d'immatriculation de votre voiture : nous l'utilisons pour l'afficher sur la page dédiée à votre voiture.

Stockage des données

Toutes les données que nous collectons sont stockées sur des serveurs situés dans l'Espace économique européen (EEE). Nous prenons des mesures de sécurité appropriées pour protéger vos informations personnelles contre toute perte, utilisation abusive ou accès non autorisé.

Utilisation des données

Nous utilisons les données collectées dans le but de :

  • Vous fournir les services de notre application web.
  • Personnaliser votre expérience utilisateur.
  • Communiquer avec vous et vous envoyer des informations relatives à notre application.

Divulgation des données

Nous ne partageons vos informations personnelles avec aucun tiers.

Vos droits

Vous avez les droits suivants concernant vos données personnelles :

  • Accès : vous pouvez demander une copie des informations personnelles que nous détenons à votre sujet.
  • Rectification : vous pouvez demander la correction ou la mise à jour de vos informations personnelles inexactes ou incomplètes.
  • Effacement : vous pouvez demander la suppression de vos informations personnelles, sous réserve de certaines exceptions.

Contactez-nous
